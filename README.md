# Crick

Playable offline-first Flutter + Flame cricket game for Android and iOS. App ID: `com.dysonc.cricket`.

Includes a 12-ball timing innings, scoring, combos, statistics, achievements, coins, daily rewards, cosmetic bats/balls/stadiums, settings, and automatic local saves.

## Windows

```powershell
git clone https://github.com/RD4U-Privy/crick.git
cd crick
Set-ExecutionPolicy -Scope Process Bypass
.\tool\setup_windows.ps1
code .
flutter run
```

Choose an Android emulator or connected device in VS Code. iOS compilation requires macOS and Xcode.
